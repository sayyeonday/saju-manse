# 사주 만세력 (四柱萬歲曆)

생년월일시를 입력하면 사주 원국과 주요 명리 정보를 계산해 보여주는 단일 페이지 웹앱입니다.
서버·빌드·외부 API 없이 브라우저에서 모든 계산이 동작합니다.

## 기능

- **사주 원국** — 시·일·월·년 4기둥(천간·지지), 오행 색상 배경
- **십성 / 12운성 / 지장간** — 일간 기준 자동 산출
- **신살** — 천을귀인·문창·양인·공망·백호·괴강·귀문·천라·지망 (기둥별 표기)
- **십이신살** — 년지·일지 기준 동시 표기
- **형충회합** — 천간합·충, 지지 육합·삼합·방합·충·형·파·해·원진
- **대운 / 세운(연운)** — 대운수·순역, 간지별 십성·12운성
- **신강·신약 / 공망 / 조후** — 지장간 통근·일수비율·위치가중 반영, 조후 희신(궁통보감 기준) 결론 표기
- **노션 연동** — 결과를 마크다운 표 + 노션 DB용 TSV로 복사

## 계산 정확도

- 절기는 천문 근사식(태양황경)으로 산출 → 년·월주 결정
- 일주는 율리우스일 기반
- **진태양시 보정** — 국가·지역 경도 + 균시차
- **한국 표준자오선 이력**(1954–1961 127.5° 등) 및 **과거 서머타임**(1948–60 일부, 1987–88) 자동 반영
- 주요국(미국·EU·호주 등) 서머타임 근사 반영
- 자시 기준(야자시/조자시) 선택 가능

> 절기 경계(±수십 분)·유파별 신살/용신 차이가 있을 수 있어 참고용입니다.
